# Password Generator

## Description
A Python application that generates secure random passwords based on user-defined length using letters and numbers.

## Features
- Generate random passwords
- User-defined password length
- Includes uppercase and lowercase letters
- Includes numbers
- Input validation

## Technologies Used
- Python
- Random module
- Lists
- Loops
- Conditional Statements

## How to Run
1. Install Python.
2. Download the project files.
3. Open terminal in the project folder.
4. Run:
python main.py
